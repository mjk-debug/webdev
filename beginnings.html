Origins of the Web

The World Wide Web (WWW) was envisioned by British computer scientist Sir Tim Berners-Lee in 1989 while working at CERN. At the time, researchers worldwide faced challenges sharing information due to incompatible computer systems. Berners-Lee proposed a system using hypertext to link and access documents across different systems, summarized in a proposal titled "Information Management: A Proposal."

Foundational Technologies

In 1990, Berners-Lee developed three critical technologies that form the basis of the modern Web:

  • HTML – the language used to create web pages
  • HTTP – the protocol used to transfer web content
  • URI – Uniform Resource Identifier, used to locate web resources

That same year, he also created the first web browser, WorldWideWeb.app, and the first web server software, httpd. The first website went live at CERN in 1991, marking the official launch of the World Wide Web.

Impact

The Web has since transformed global communication, information sharing, and technology itself, setting the stage for continuous innovation and digital transformation.
