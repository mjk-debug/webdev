From Mosaic to Chrome

The first web browser was WorldWideWeb.app, created by Tim Berners-Lee. However, the first popular graphical browser was Mosaic, released in 1993 by Marc Andreessen and NCSA. Mosaic introduced the graphical user interface (GUI), allowing text and images to be viewed together.

Mosaic later evolved into Netscape Navigator, which became a dominant browser in the early 90s. Competition grew fierce when Microsoft launched Internet Explorer in 1995, bundled with Windows OS.

In 2004, Mozilla Firefox introduced tabbed browsing and enhanced security. In 2008, Google released Chrome, emphasizing speed, simplicity, and security. It soon became the most widely used browser. Other browsers such as Safari, Opera, and Microsoft Edge continue to serve niche and general audiences.
